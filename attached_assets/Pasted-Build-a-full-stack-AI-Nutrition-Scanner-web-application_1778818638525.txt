Build a full-stack AI Nutrition Scanner web application called "NutriScan AI" that analyzes food images using computer vision and returns detailed nutritional data with health recommendations.

---

## TECH STACK

Frontend: React.js + Tailwind CSS
Backend: Node.js + Express
AI/ML: Anthropic Claude API (claude-sonnet-4) with vision
Database: MongoDB (store scan history)
File Upload: Multer (image handling)
Deployment: Vercel (frontend) + Railway (backend)

## CORE FEATURES TO BUILD

### 1. Image Upload System
- Drag & drop zone with visual feedback
- Click to browse file picker
- Preview uploaded image before analysis
- Validate: JPG, PNG, WEBP only, max 5MB
- Show upload progress indicator

### 2. AI Vision Analysis (Claude API)
Send the image to Claude claude-sonnet-4 with this system prompt:

"You are a certified nutritionist and food scientist with expertise in 
 computer vision food recognition. Analyze the food image provided and 
 return accurate nutritional data as a strict JSON object only."

Request this JSON structure back:
{
  "foodName": "string",
  "confidence": 0-100,
  "servingSize": "string",
  "calories": number,
  "protein": "Xg",
  "carbs": "Xg",
  "fat": "Xg",
  "fiber": "Xg",
  "sugar": "Xg",
  "sodium": "Xmg",
  "cholesterol": "Xmg",
  "healthScore": 0-100,
  "healthSummary": "string",
  "vitamins": [{ "name": "string", "amount": "string", "dailyPct": number }],
  "minerals": [{ "name": "string", "amount": "string" }],
  "allergens": ["string"],
  "dietaryFlags": ["Vegan", "Gluten-Free", "Keto-Friendly", etc],
  "ingredients": ["string"],
  "recommendations": [{ "emoji": "string", "text": "string", "type": "positive|warning|tip" }]
}

### 3. Nutrition Dashboard UI
Display results in a visually rich dashboard:
- Large calorie display with daily % ring chart
- Macro breakdown: Protein / Carbs / Fat as bar charts
- Micronutrients table (vitamins & minerals)
- Health Score ring (0-100, color coded green/yellow/red)
- Dietary badges (Vegan, Keto, Gluten-Free, etc.)
- Detected allergen warning tags
- 4-6 personalized health recommendations

### 4. Charts & Visualizations
Use Recharts library:
- PieChart for macro distribution
- RadialBarChart for health score
- BarChart for vitamin daily percentages
- Use smooth animations on load

### 5. Scan History
- Save each scan to MongoDB
- Show last 10 scans in a grid
- Click any past scan to view full results
- Show thumbnail + food name + calories
- Add delete functionality

### 6. Meal Tracker (Bonus)
- Let users add scans to "Today's Meals"
- Calculate running daily totals
- Show remaining daily calorie budget
- Persist in localStorage

## BACKEND API ENDPOINTS

POST /api/nutrition/analyze
- Accepts: multipart/form-data (image file)
- Converts image to base64
- Sends to Claude API with vision prompt
- Parses JSON response
- Saves to MongoDB
- Returns: nutrition JSON

GET /api/nutrition/history
- Returns last 20 scan results from DB

DELETE /api/nutrition/:id
- Deletes a scan from history
